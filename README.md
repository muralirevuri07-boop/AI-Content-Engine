# AI Content Engine

An automated multi-agent system that generates complete content pipelines — from idea to final output.

---

## What This Does

This project simulates a real-world AI content production workflow using multiple specialized agents:

* Script Agent → Generates content ideas & scripts
* Visual Agent → Creates images/visuals
* Voice Agent → Adds AI voice narration
* Subtitle Agent → Generates captions
* Assembly Agent → Combines everything into final output

---

## Tech Stack

* Python
* OpenAI API
* AI Agent Architecture
* Modular Multi-Agent Design

---

## Why This Project?

Most AI apps are single-function tools.

This project explores how **multiple AI agents can collaborate** to automate real workflows — similar to how teams operate in companies.

## How to Run

```bash
git clone https://github.com/your-username/AI-Content-Engine.git
cd AI-Content-Engine
pip install -r requirements.txt
python app.py
```

---

## Environment Setup

Create a `.env` file:

```env
OPENAI_API_KEY=your_key_here
```

⚠️ Never upload your real API keys to GitHub.
